
# AI Interview Analyzer

## 📌 Project Description
This project analyzes interview performance using AI. It takes voice input, converts it to text, analyzes answers, detects emotions, and generates a final score.

## 🚀 Features
- Voice-based interview questions
- Speech-to-text conversion
- Answer evaluation
- Emotion detection using camera
- Voice confidence analysis
- Final score generation

## 🛠️ Technologies Used
- Python
- OpenCV
- Speech Recognition

## ▶️ How to Run
1. Install Python
2. Install required libraries:
   pip install -r requirements.txt
3. Run the project:
   python main.py
